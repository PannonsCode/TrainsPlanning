INSERT THE NAME OF THE TRAIN, ITS INITIAL AND ITS FINAL POSITION

RULES:

-The name of the train is a single string

-The initial and final position can be a station or a middle rail:

o If the initial position is a station insert "name_rail", where 'name' is the name of the station and 'rail' is the number of the rail. (Ex. A_1, B_3, Roma_2, ecc.)

o If the final position is a station insert only "name", where 'name' is the name of the station. (Ex. A, B, Roma, ecc.)

o If the initial and/or final position is a middle rail insert "node1,node2", where 'node1' and 'node2' are the names of the nodes between which the rail is included.

-Insert all the trains you want pressing the "ADD" button for each one, a recap is in the text area below

-Press the "SUBMIT" button to download the file with inserted data and reset the form
